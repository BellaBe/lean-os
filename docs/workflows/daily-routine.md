# Daily Routine

Human touchpoint: <30 min/day on operations.

## Morning Review (5 minutes)

### Read ops/today.md

Auto-generated daily summary:

**High-priority items (human approval required):**
- Content opportunities (priority ≥0.7)
- Content drafts ready for review
- Content ready to publish
- Sales calls scheduled
- Strategic decisions flagged

**Decisions made by AI (last 24h):**
- Marketing activities
- Sales activities
- Canvas updates

**Alerts:**
- Performance anomalies
- Process bottlenecks
- Pattern deviations

---

## Human Actions (3-5 minutes)

1. **Approve opportunities** (30 sec) - Approve/defer/reject flagged items
2. **Review drafts** (1-2 min) - Check technical accuracy, brand voice
3. **Publish content** (1 min) - Publish approved content, add URLs
4. **Review call prep** (30 sec) - Skim pitch deck, note talking points

---

## AI Handles Autonomously

- Lead qualification
- Content generation
- Pipeline updates
- Performance tracking
- Pattern detection
- Follow-up emails
- Prospect research

---

## Weekly Review (Optional, 30 min)

- Review performance metrics
- Check velocity.md for bottlenecks
- Review patterns.md for insights
- Adjust priorities if needed

---
