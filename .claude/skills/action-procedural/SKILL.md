# Procedural Action Skill

## Purpose
Create a repeatable, deterministic process.

## When to use
- SOPs
- Runbooks
- Operational checklists

Do NOT use for:
- one-off plans (use planning-action)
- recommendations (use prescriptive-actions)

## Operating rules
1. Define trigger conditions.
2. List prerequisites.
3. Provide numbered steps with expected outcomes.
4. Include decision points and escalation.
5. Include rollback/cleanup steps.

## Outputs
### Purpose & triggers
### Prerequisites
### Procedure steps
### Decision points / escalation
### Rollback / cleanup
### Checklist
