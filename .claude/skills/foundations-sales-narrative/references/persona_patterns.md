# Persona Narrative Patterns

## Economic Buyer Example

### Problem
{Company} currently spends $150K/year on manual {process}:
- 3 staff × 20 hrs/week × $50/hr = $150K annually
- Opportunity cost: Only 50 {outputs}/day vs potential 200/day
- Competitive risk: AI-powered competitors launching 2x faster

### Solution
{Your product} AI replaces manual workflow:
- 60% time reduction = $90K annual savings
- 4x {output} capacity = +$300K potential revenue
- Payback period: 3.2 months

### Specifics
**ROI Model**:
- Cost: $30K/year subscription
- Savings: $90K/year in staff time
- New revenue: $300K from increased {capability}
- Net benefit: $360K year 1
- Payback: 3.2 months

**Risk Mitigation**:
- 30-day free trial (no commitment)
- Case study: {Similar company} achieved 65% time reduction
- Month-to-month after first year

## Technical Buyer Example

### Problem
Current manual process creates technical debt:
- No API integration (staff use spreadsheets)
- Inconsistent data formats
- No version control on {outputs}
- Manual QA catching 40% error rate

### Solution
{Your product} provides clean API integration:
- REST API with OpenAPI spec
- Webhooks for real-time updates
- Built-in validation (reduces errors to <5%)
- SOC2 Type II compliant

### Specifics
**Implementation Timeline**:
- Week 1: API keys, sandbox access
- Week 2: Integration development
- Week 3: Testing, QA
- Week 4: Production deployment

**Technical Specs**:
- 99.9% uptime SLA
- <100ms p95 latency
- Rate limits: 1000 req/min
- Data residency: US/EU options

## End User Example

### Problem
User workflow is frustratingly slow:
- 30 minutes per {task} (research, manual curation)
- Inconsistent results
- Redoing work when customer rejects output

### Solution
{Your product} makes {task} instant:
- 30 seconds per {task} (60x faster)
- AI learns preferences
- Smart alternatives when customer says "not quite"

### Specifics
**Daily Impact**:
- Before: 10 {outputs}/day (5 hours)
- After: 100 {outputs}/day (1 hour)
- Time saved: 4 hours/day for other work

**Learning Curve**:
- Day 1: Basic features (15 min training)
- Week 1: Advanced features mastered
- Month 1: Custom configurations
