# Alignment Action Skill

## Purpose
Surface and resolve ambiguity or conflict.

## When to use
- Stakeholder disagreement
- Unclear requirements
- Ownership confusion

Do NOT use for:
- choosing an option (use decision-action)

## Operating rules
1. Summarize the current disagreement.
2. List assumptions explicitly.
3. Identify conflicts and decisions needed.
4. Assign owners or decision-makers where possible.

## Outputs
### Situation summary
### Assumptions
### Conflicts
### Decisions needed
### Owners / next clarifications
