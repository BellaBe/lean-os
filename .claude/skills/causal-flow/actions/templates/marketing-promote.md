# Marketing Promote Template

**Action Type:** `marketing:promote`

## Objective
Amplify content reach through paid promotion and outreach.

## Process (AI-automated + human budget approval, 14-30 days)

### 1. Paid Promotion Setup (1 day)

**LinkedIn Ads:**
- [ ] Campaign created (awareness/conversion)
- [ ] Targeting: {job titles, companies, industries}
- [ ] Budget: ${daily} per day for {days} days
- [ ] Ad creative: {post or carousel}

**Retargeting:**
- [ ] Website pixel installed
- [ ] Retargeting audience created (visitors)
- [ ] Budget allocated

### 2. Organic Amplification (AI-automated)

**Employee Advocacy:**
- [ ] Share requests sent to team
- [ ] Amplification tracking

**Community Engagement:**
- [ ] Relevant LinkedIn groups identified
- [ ] Content shared to groups
- [ ] Comments and discussions monitored

### 3. Direct Outreach (AI-assisted)

**Warm Leads:**
- [ ] 50-200 personalized messages (LinkedIn DMs)
- [ ] Message template: "{Name}, thought you'd find this relevant: {link}"
- [ ] Response tracking

### 4. Budget Monitoring (AI-automated)

**Daily Tracking:**
- Spend vs budget
- CPC, CPM, CTR
- Auto-pause if overspend

## Success Criteria
- [ ] Budget approved and released
- [ ] Ads launched on schedule
- [ ] Daily monitoring active
- [ ] Auto-pause at budget limit

## Output Format
```markdown
# Promote: {Campaign Name}
**Duration:** {start} - {end}
**Budget:** ${total}

## Paid Promotion
- LinkedIn ads: ${spend}, {impressions} impressions, {CTR}% CTR
- Retargeting: ${spend}, {conversions} conversions

## Organic Reach
- Employee shares: {count}
- Community posts: {count}

## Outreach
- Messages sent: {count}
- Responses: {count} ({percent}%)
```
