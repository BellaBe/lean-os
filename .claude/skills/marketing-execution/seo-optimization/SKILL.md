# SEO Optimization (Optional)

Apply basic SEO for long-tail discoverability. Low priority — loops matter more.

## When to Use

**Use for:**
- Evergreen content (guides, documentation)
- High-value long-tail keywords
- Content already performing (optimization boost)
- Explicit request

**Don't prioritize for:**
- Time-sensitive content
- Loop-focused content (social-first)
- Low search volume topics

## Quick Optimization Checklist

### Title (Required)
- [ ] Primary keyword included
- [ ] 50-60 characters
- [ ] Compelling (not just keyword)

### Meta Description (Required)
- [ ] 150-160 characters
- [ ] Includes keyword
- [ ] Includes benefit/result

### Headings (Required)
- [ ] One H1 (title)
- [ ] 3-7 H2s with keyword variations
- [ ] Logical structure

### Content (Required)
- [ ] Keyword in first 100 words
- [ ] 0.5-2% keyword density
- [ ] Natural integration (no stuffing)

### Internal Links (Required)
- [ ] 3-5 relevant internal links
- [ ] Descriptive anchor text

### URL (Required)
- [ ] Keyword in slug
- [ ] Short (3-5 words)
- [ ] Hyphens, lowercase

### Images (If Present)
- [ ] Alt text with keyword
- [ ] Descriptive filename
- [ ] Compressed (<200KB)

## Remember

**SEO is dying channel.** Elena Verna data: G2 lost 80-90% traffic since ChatGPT.

Optimize if it's easy. Don't prioritize over loop mechanics.

**Loop triggers > SEO optimization** for content success.
