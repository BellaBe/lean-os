# Marketplace Copy Reference

Platform-specific guidelines for app store/marketplace content.

## Shopify App Store

| Element | Limit | Best Practice |

| Title | 30 chars | Keyword + benefit |
| Tagline | 70 chars | Primary value prop |

### Keywords

- Primary keyword in title
- Secondary keywords in tagline and first paragraph
- Natural usage in description (no stuffing)

### Screenshots

| Position | Purpose | Content |
|----------|---------|---------|
| 1 | Hero | Main value prop |
| 2 | Problem/Solution | Before ‚Üí After |
| 3 | Key feature | In context |
| 4 | Results | Metrics |
| 5 | Setup | Ease of use |

### Review Prompts

**Soft ask (after first value):**
```
Quick question: Is {app} helping your customers?
If yes, a review helps other merchants find us.
[Leave review] [Maybe later]
```

**Direct ask (after repeated value):**
```
You've had {X} customers engage with {feature} this week.
If we're adding value, we'd love a review.
[Leave review]
```

**Celebration (after milestone):**
```
üéâ {Milestone achieved}!
Would you share your experience?
[Leave review]
```

### Review Responses

| Rating | Approach |
|--------|----------|
| 5 stars | Thank, highlight what they mentioned, invite feedback |
| 4 stars | Thank, ask what would make it 5 |
| 3 stars | Thank, ask for specifics, offer support |
| 1-2 stars | Apologize, take offline, fix it |

**Response SLA:** 24 hours, always.

## General Marketplace Principles

### Headlines

```
{Primary keyword} {benefit} for {audience}
```

**Good:** "AI Product Recommendations for Fashion Stores"
**Bad:** "GlamYouUp - The Best App"

### Benefits vs Features

| Feature | Benefit |
|---------|---------|
| AI-powered | Get personalized recommendations |
| Real-time sync | Always up-to-date inventory |
| One-click install | Start in 2 minutes |

Always lead with benefit.

### Social Proof

Include in description:
- Number of users/installs
- Star rating (if good)
- Notable customers (if allowed)
- Specific metrics ("40% fewer returns")

### Support Section

Always include:
- Response time commitment
- Help documentation link
- Contact method
- Hours/timezone
